## 一、账号管理与登录

### 1.1 账号体系

本系统支持三种角色账号：

| 角色 | 说明 | 注册后状态 |
|------|------|-----------|
| **学生** (student) | 普通学生用户 | 自动启用 |
| **学校管理员** (school_admin) | 就业指导中心人员 | 自动启用 |
| **企业** (company_admin) | 用人单位 | 需管理员审核 |

### 1.2 管理员账号

#### 创建管理员

使用 CLI 工具创建管理员账号：

```bash
cd backend

# 创建管理员（不指定密码使用默认密码 Admin123!）
python -m app.cli admin --username admin --name "系统管理员"

# 指定密码创建
python -m app.cli admin --username admin --name "系统管理员" --password "YourPassword123!"
```

#### 管理员登录

1. 访问 `/login` 页面
2. 输入用户名和密码（默认：`admin` / `Admin123!`）
3. 选择角色为系统管理员
4. 点击登录

### 1.3 账号安全机制

- **登录限流**：同一IP每分钟最多5次，同一用户名每分钟最多10次
- **账号锁定**：连续5次密码错误锁定15分钟
- **Token刷新**：使用Refresh Token Rotation机制
- **多设备管理**：支持查看和终止其他会话

### 1.4 企业账号审核

企业注册后需要管理员审核才能登录：

```bash
# 查看待审核企业
python -m app.cli list-companies

# 审核通过
python -m app.cli verify-company --company-id C001 --action approve

# 审核拒绝
python -m app.cli verify-company --company-id C001 --action reject
```

## 三、学生端功能 (路径: /student/*)

### 3.1 个人首页 (/student/dashboard)

- 欢迎卡片（显示姓名、学院、专业）
- 就业状态卡片（待就业/已签约/升学/出国）
- AI竞争力评分雷达图
- 推荐的岗位列表（3-5条）
- 求职进度时间线

### 3.2 档案管理 (/student/profile)

- 查看/编辑个人信息
- 上传简历（PDF/图片）
- 完善求职意向（城市、行业、薪资）
- 查看档案完整度

### 3.3 岗位推荐 (/student/jobs)

- 筛选：城市、行业、薪资、学历
- 搜索：关键词搜索
- 列表：岗位卡片（公司名、职位、薪资、发布时间）
- 详情：弹窗查看岗位详情+投递按钮

### 3.4 AI就业画像 (/student/ai-profile)

- 输入：专业、GPA、技能、目标城市、实习经历
- 输出：就业竞争力评分(0-100)
- 雷达图：专业匹配度、技能契合度、地区供需比、薪资预期合理性
- 优势/劣势分析文字
- 改进建议

### 3.5 AI简历优化 (/student/ai-resume)

- 简历文本输入框（富文本）
- 目标岗位输入
- AI分析：ATS评分、关键词匹配度、缺失关键词
- 优化建议列表

### 3.6 AI考研vs就业决策 (/student/ai-decision)

- 输入：目标城市、期望薪资、备考时间
- 输出：考研回报率分析
- 可视化：两条路径对比图

### 8.1 技术栈

| 层级 | 技术 | 说明 |

- AI服务：ai_services, ai_analysis_requests, ai_recommendations

## 九、快速开始

### 9.1 启动后端

```bash
cd backend

# 激活conda环境
conda activate cccc

# 初始化数据库（如需要）
python -m app.cli init-db
python -m app.cli create-roles

# 创建管理员
python -m app.cli admin --username admin --name "系统管理员"

# 启动服务
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 9.2 启动前端

```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 十、常见问题

### Q1: 企业账号无法登录？
企业注册后需要管理员审核。请联系系统管理员执行审核命令。

### Q2: 忘记管理员密码？
可以使用CLI重新创建一个管理员账号：
```bash
python -m app.cli admin --username admin2 --name "新管理员"
```

### Q3: 如何解锁被锁定的账号？
```bash
python -m app.cli unlock --account-id 账户ID
```

### Q4: 如何查看所有用户？
可以通过数据库查询：
```sql
SELECT * FROM accounts;
```
